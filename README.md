### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ git clone https://gitlab.com/saprince93/assessment-test.git
$ cd [folderName]
$ npm install
$ npm start
```

For production environments...

```sh
$ npm install
$ npm run build
```
